Typical Backbone Application
============================

---

## Backbone.JS is great

* Low level
* Very flexible
* Clean

<p style="margin:40px 0;">
But it requires a lot of boilerplate.
</p>

---

## Kernel

The application main entry point

1. Load dependencies
2. Get remote data
3. Register routes
4. Setup UI
5. Show loading
6. Wait for data
7. Start router

---

## Controller

1. Show loading
2. Get remote data
3. Wait for data
4. Create and display view(s)
